BallsAndSticksModeOptions

Balls and Sticks mode options.

Properties:
Name Type Description
atom number

Sphere radius as a fraction of Van der Waals atom radius.
bond number

Cylinder radius in angstroms.
space number

Fraction of the space around one cylinder. Zero means that cylinder fills all available space.
multibond boolean

Toggles rendering of multiple ordered bonds.
aromrad number

Minor radius of a torus for aromatic loops.
showarom boolean

Toggles rendering of aromatic loops.
polyComplexity PolyComplexity

Polygonal complexity settings for different resolutions.

Properties:
Name Type Description
