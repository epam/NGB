Class: LoaderList

LoaderList

A list of available loaders.

Constructor

new LoaderList(someLoadersopt, non-null)

Create a list of loaders. The loaders are indexed by supported source types (.types property of a Loader subclass). The loaders can be retrieved later by matching against specs (see LoaderList#find).

Parameters:
Name Type Attributes Description
someLoaders Array.<function(new:Loader)> <optional>

A list of Loader subclasses to automatically register at creation time.
