removeEventListener(typeopt, nullable, callbackopt, contextopt)

Removes a previously-bound callback function from an object. If no context is specified, all versions of the callback with different contexts will be removed. If no callback is specified, all callbacks of the type will be removed. If no type is specified, callbacks for all events will be removed.
